## WordPress docker demo

### apple silicon Mac

- Go to the wordpress folder under the project

- Go to the arm folder under the wordpress folder

- Just run the command

```shell
    docker-compose up --build
```

### intel silicon Mac

- Go to the wordpress folder under the project

- Just run the command

```shell
    docker-compose up --build
```

## Intro

- docker-compose demo
- 项目目前仅在 Mac 电脑上运行通过，其他环境需自行调试，若要支持其他环境，欢迎提交 PR

- docker 环境搭建请 google 检索完成

## WordPress docker demo

### apple silicon Mac

- 进入项目的 wordpress 文件夹

- 进入 arm 文件夹

- 运行如下命令即可：

```shell
    docker-compose up --build
```

### intel silicon Mac

- 进入项目的 wordpress 文件夹

- 运行如下命令即可：

```shell
    docker-compose up --build
```
